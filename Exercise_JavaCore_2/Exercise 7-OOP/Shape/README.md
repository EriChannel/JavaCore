## Bài tập
Xây dựng lớp abstract Hình học, tạo hai lớp Hình tròn và hình chữ nhật kế thừa lớp hình học
In thông tin các hình bao gồm chu vi, diện tích, đường kính(đối với hình tròn), đường chéo(đối với hình chữ nhật)
